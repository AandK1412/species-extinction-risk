# Data — Species Extinction Risk Dataset

`species_extinction_risk.csv` is a **synthetic** dataset (1,885 records)
regenerated by `../src/generate_dataset.py`. The team's original AI-generated
dataset was not shipped with the report, so this is a faithful reconstruction
that matches the documented schema, distributions, and relationships. Because it
is randomly generated, exact values differ from the report, but the patterns and
model conclusions are reproduced.

## Schema (from Table 2.1.1 of the report)

| Column | Description | Type |
|---|---|---|
| `Species` | Common name of the species | Categorical |
| `Region` | Geographic region where the species resides | Categorical |
| `Pollution_Level` | Pollution index in the habitat (0–100) | Numeric |
| `Illegal_Wildlife_Trade_Rate` | Pressure from illegal trade (0–100) | Numeric |
| `Food_Availability` | Availability of food in the habitat (0–100) | Numeric |
| `Human_Population_Density` | People per km² near the species' region | Numeric |
| `Species_Reproduction_Rate` | Births per year | Numeric |
| `Climate_Change_Impact` | % impact of climate change on the environment | Numeric |
| `Protected_Area_Presence` | In a protected conservation area (0/1) | Binary |
| `Invasive_Species_Presence` | Invasive species present (0/1) | Binary |
| `Extinction_Risk` | **Target** — species classified as at risk (0/1) | Binary |

## Built-in relationships

The target is generated as a noisy logistic function of the predictors so that
the analysis reproduces the report's findings:

- **Pollution Level** — strongest positive driver of extinction risk.
- **Human Population Density**, **Illegal Wildlife Trade Rate** — strong positive.
- **Climate Change Impact**, **Invasive Species Presence** — moderate positive.
- **Protected Area Presence** — negative (protection lowers risk).
- **Food Availability**, **Species Reproduction Rate** — weak / not significant.

Random noise is calibrated so models reach roughly the report's AUC (~0.70).
